## 📂 Project Logs

### 🏗️ Project 1: Flexbox & Grid Centering
* **Description:** A practical exploration of centering elements perfectly using CSS Flexbox and Grid, along with implementing smooth animations.
* **Key Concepts:** * Perfect centering with Flexbox (`display: flex`, `justify-content: center`, `align-items: center`).
  * Center alignment with CSS Grid (`display: grid`, `place-items: center`).
  * Creating fluid motion using CSS `@keyframes` and transitions.

### ⚽ Project 2: Carousel Effect (World Cup Edition)
* **Description:** A dynamic, interactive image/content carousel themed around the World Cup.
* **Key Concepts:** Overflow control, CSS transitions, and positioning.

### ⏳ Project 3: Loader Window Animation
* **Description:** A clean, CSS-only loading animation designed for web application splash screens.
* **Key Concepts:** Keyframe animations, infinite loops, and transform properties.

### 🔑 Project 4: Password Generator UI
* **Description:** A clean and modern user interface for a password generator application.
* **Key Concepts:** Form styling, semantic HTML inputs, and component layout.

### 🧮 Project 5: Calculator UI
* **Description:** A sleek, responsive user interface for a standard calculator.
* **Key Concepts:** Grid layout for button alignment, hover states, and active button effects.

### 💡 Project 6: Newton's Cradle Animation
* **Description:** A physics-based pendulum animation built entirely out of CSS shapes.
* **Key Concepts:** Advanced use of `::before` and `::after` pseudo-elements, `transform-origin`, and precise animation timing functions.

### 🃏 Project 7: Advanced Card Layout (World Cup Edition)
* **Description:** A premium card component layout featuring advanced hover effects and responsive grids.
* **Key Concepts:** CSS Grid layout, multi-layered shadows, clip-paths, and micro-interactions.
