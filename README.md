## 📂 Project Logs

### 🏗️ Project 1: Demonstration of Flex and Grid properties to align content in center
* **Description:** A practical exploration of centering elements perfectly using CSS Flexbox and Grid, along with implementing smooth animations.
* **Key Concepts Learned:** 
  * Perfect centering with Flexbox (`display: flex`, `justify-content: center`, `align-items: center`).
  * Center alignment with CSS Grid (`display: grid`, `place-items: center`).
  * Creating fluid motion using CSS `@keyframes` and transitions.
